# 📊 Association Rules Analysis – Supermarket Data Mining

This repository contains the materials developed for the **Data Mining** course. The objective of this project was to analyze the association between supermarket products using **association rule mining techniques**, generating scenarios and possible business interventions.

---

## 🗂️ Repository Contents
📓 Python notebook with the data analysis process (Jupyter Notebook / Google Colab)  
📄 (Optional) Report with explanations, metrics, and interpretations of the generated rules  

---

## 🔍 Analysis Focus
The project focuses on identifying relationships between products in a supermarket dataset, such as:

- Frequently purchased product combinations  
- Associations between items  
- Consumer behavior patterns  

---

## 🧪 Methodology
- Data preprocessing and cleaning  
- Transformation of data into transactional format  
- Application of association rule algorithms (Apriori)  
- Evaluation using key metrics:

  - Support  
  - Confidence  
  - Lift  

The analysis was performed using **Python**, with the following libraries:

- Pandas  
- NumPy  
- mlxtend  
- Matplotlib / Seaborn  

---

## 🎯 Objective
To discover meaningful relationships between products and generate insights that support decision-making, such as product placement, promotions, and sales strategies.

---

## 📊 Key Concepts
- **Support**: Frequency of occurrence of an itemset in the dataset  
- **Confidence**: Probability that a rule is valid  
- **Lift**: Strength of a rule compared to random chance  

---

## 📈 Results
The analysis revealed important associations between products, allowing the creation of scenarios and possible interventions, such as:

- Product bundling strategies  
- Shelf organization optimization  
- Targeted promotions
